# Gojo Satoru Moveset Mod

A Fabric mod for Minecraft 1.21.1 that adds Gojo Satoru's abilities from Jujutsu Kaisen.

## Abilities

### Infinity
- **Command**: `/gojo infinity`
- **Keybind**: I
- **Effect**: Grants near-invulnerability with Resistance V and slight slowness
- **Visual**: Enchant particles surround the player

### Limitless
- **Command**: `/gojo limitless`
- **Keybind**: O
- **Cooldown**: 10 seconds
- **Effect**: Creates a powerful explosion that damages and pushes back nearby entities
- **Visual**: Dragon breath particles and explosion effect

### Domain Expansion: Unlimited Void
- **Command**: `/gojo domain`
- **Keybind**: P
- **Cooldown**: 60 seconds
- **Duration**: 15 seconds
- **Effect**: Affects all entities in a large radius with blindness, slowness, and weakness
- **Player Buff**: Strength V and Speed II
- **Visual**: Massive portal particle circle around the player

## Installation

1. Build the mod using `./gradlew build`
2. Copy the resulting jar from `build/libs/` to your `mods` folder
3. Install Fabric Loader for Minecraft 1.21.1

## Usage

- Use commands or keybinds to activate abilities
- Check the HUD for ability status and keybinds
- Abilities have cooldowns to prevent spam

## Configuration

Cooldowns and damage values can be modified in the GojoAbilityManager class.
